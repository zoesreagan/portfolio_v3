Eidetic: A Working Memory Game

A visual memory game built using HTML, JS + CSS for General Assembly - Fullstack Development Bootcamp.

Introduction

Visuospatial processing diminishes with age and the loss of this skill is a major component of cognitive decline. Perhaps training this "memory" can help ward off the loss of visuospatial processing we experience with aging. This game tests the player's ability to recall a word after the image matching portion.



Setting up the game board

I created low fidelity wireframes to help organize game logic and layout. I later used Balsalmiq to organize a desktop view with a mid fidelity wireframe.



Building the Game

Eidetic is constructed primarily with vanilla JavaScript, with some jQuery to navigate the DOM. There are 16 selectable divs and the code logic checks if the cards match. If so, the div is assigned a 'match' class and is removed from view. The game also employs a number of modals, which asks the player to remember, and then recall the word from three available options.
